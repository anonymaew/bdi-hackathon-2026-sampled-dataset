# BDI Hackathon 20226 Sampled Dataset

ชุดข้อมูลตัวอย่างสำหรับแการข่งขัน BDI Hackathon แบ่งเป็น 7 Dataset 3 Track

## Data Overview

| ชื่อชุดข้อมูล | Track | คำอธิบาย |
|-------------|-------|----------|
| [`ชุดข้อมูลสเปกตรัม NMR สำหรับตรวจรูปแบบสารเคมี`](nmr-pattern/) | **Phenome** | สัญญาณ NMR สองมิติ 20,000+ คอลัมน์/ตัวอย่าง — ใช้ระบุชนิดและตำแหน่งสารประกอบทางเคมี |
| [`ชุดข้อมูลสารเมตาบอไลต์ NMR สำหรับเปรียบเทียบผู้ป่วย`](nmr-result/) | **Phenome** | สารเมตาบอไลต์ abundance + sample table เปรียบเทียบผู้ป่วย vs. กลุ่มควบคุม |
| [`ชุดข้อมูล EMR ผู้ป่วยความดันแบบติดตามยาว`](hypertension/) | **Medical** | EMR ผู้ป่วยความดันโลหิตสูง ~150,000 คน — สัญญาณชีพ, Lab, Comorbidity, ยา ตามช่วงเวลา |
| [`ชุดข้อมูล EMR ผู้ป่วยเบาหวานแบบติดตามยาว`](diabetes/) | **Medical** | EMR ผู้ป่วยเบาหวาน ~70,000 คน — ข้อมูลผู้ป่วย, สัญญาณชีพ, Lab, Comorbidity, ยา |
| [`ชุดข้อมูลคลื่นของเครื่องช่วยหายใจจากผู้ป่วย ICU`](ventilator/) | **Medical** | คลื่น ventilator — Flow, Pressure, Volume rate 25Hz, 24 ช่วง/วัน, 2–3 สัปดาห์ |
| [`ชุดข้อมูลสแกน 3 มิติ เมืองขอนแก่น`](3dpoints/) | **Smart City** | Point Cloud / Mesh ครอบคลุมพื้นที่เมือง — แบ่งตาม Tile พิกัด สำหรับวิเคราะห์โครงสร้าง |
| [`ชุดข้อมูลบันทึกคำร้องจากประชาชน เทศบาลนครขอนแก่น`](complaints/) | **Smart City** | คำร้อง/ร้องเรียน ~46,000 รายการ — ตั้งแต่รับเรื่องจนดำเนินการเสร็จสิ้น |
